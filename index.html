Open source

  • alinea

    A headless content management system for JavaScript runtimes, written in TypeScript.

  • esbx

    A collection of esbuild plugins (browser reloads, running commands on successful builds, sass and css modules, legacy compilation to ES5, ...).

  • genes

    An alternative backend for the Haxe compiler which generates ES6 modules and Typescript definitions.

  • smtpmailer

    An implementation of the SMTP protocol for the Haxe programming language with support for SSL and StartTLS.

  • asys

    An abstraction over the Haxe I/O standard library enabling a unified asynchronous API for both static targets and Node.js.

  • unit

    A simple and fast test runner for Haxe.

  • jdiff

    An implementation of json diff RFC6902 (JSON Patch) for Haxe.

  • graphql

    Implementation of GraphQL for Haxe, leveraging the established PHP and JavaScript libraries. Includes macro utilities to generate a full schema based on compile time types.

  • pixels

    Utilities for Haxe to work with pixel byte arrays. Includes ports of an image resampler and smart crop detection.

  • monsoon

    An express inspired, minimal web framework and embedded webserver for Haxe.
